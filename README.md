# STEAM Landing Page

Landing page STEAM được xây dựng bằng React + Vite, tối ưu cho bài nộp có yêu cầu tách branch feature, commit chuẩn và pull request rõ ràng.

## Run project

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Required feature branches

- `feature/setup-project`
- `feature/header`
- `feature/hero`
- `feature/course`
- `feature/testimonial`
- `feature/form`
- `feature/footer`
- `feature/responsive`

## Pull request format

Title example:

```text
[Feature] Hero
```

Description example:

```text
- Add hero
- CTA button
```
